# Command quick reference

[← Back to README](../README.md)

## Auth

- `npmctl login`
- `npmctl login-status`
- `npmctl logout`

## Secrets

- `npmctl cf-token-set`
- `npmctl cf-token-status`
- `npmctl cf-token-delete`
- `npmctl unifi-api-key-set`
- `npmctl unifi-api-key-status`
- `npmctl unifi-api-key-delete`

## UniFi DNS

- `npmctl list-unifi-sites`
- `npmctl add-unifi-dns-record`
- `npmctl add-new-app`

## Nginx Proxy Manager

- `npmctl add-cert-cloudflare`
- `npmctl add-proxy-host`
- `npmctl add-proxy-with-cert`

## Diagnostics

- `npmctl doctor`

## Handy help commands

```bash
npmctl --help
npmctl <command> --help
```
